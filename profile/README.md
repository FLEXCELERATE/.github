# FLEXCELERATE Solutions GmbH

**Modular process automation for the pharmaceutical and chemical industries.**

We're a Dresden-based engineering company that helps manufacturers plan faster, build more efficiently, and meet regulatory requirements through modularization. Our work spans MTP (Module Type Package) development, ISA-88 recipe engineering & BatchML, VDI/VDE 3682 formalized process descriptions, DEXPI, process simulation, and virtual commissioning.

### Open Source

**[fpd-editor](https://github.com/FLEXCELERATE/fpd-editor)** — A text-based editor for VDI/VDE 3682 Formalized Process Descriptions. Write process descriptions in a PlantUML-inspired syntax, get standards-compliant diagrams in real time. Python + React/TypeScript, MIT licensed.
